# Sales Performance Analysis Dashboard using Power BI

## Project Overview
This project analyzes sales performance using Power BI to provide business insights on sales, profit, and trends across regions and products.

## Tools Used
- Power BI
- Excel
- Power Query
- DAX

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin

## Key Insights
- East region generated the highest sales
- Maharashtra contributed the maximum profit
- Electronics category was the most profitable
- Identified top-performing products and salesperson

## Project Type
Academic / Guided Project completed during a learning program using a simulated business dataset.
